## APIs

| Version | Resource                                                                                                                                                                          | Description                                                                                    |
| ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| V2      | <a className="nowrap" href="api/tasks"><VscJson className="devicons" fontSize="large" /> switchboard-tasks</a>                                                                    | Protobuf definitions for the various supported task definitions to build job definitions from. |
| V2      | <a className="nowrap" href="api/switchboardv2-api"><SiTypescript className="devicons" fontSize="large" /> switchboardv2-api</a>                                                   | A typescript library of utility functions to interact with the Switchboardv2 program.          |
| V2      | <a className="nowrap" href="api/switchboardpy"><SiPython className="devicons" fontSize="large" /> switchboardpy</a>                                                               | A python library of utility functions to interact with the Switchboardv2 program.              |
| V2      | <a className="nowrap" href="api/switchboardv2-cli"><SiPowershell className="devicons" fontSize="large" /> switchboardv2-cli</a>                                                   | A Command Line Interface (CLI) to interact with the Switchboardv2 program.                     |
| V2      | <a className="nowrap" href="https://docs.rs/switchboard-v2"><SiRust className="devicons" /> switchboard-v2 <GoLinkExternal className="devicons" /></a>                            | A rust library of utility functions to interact with the Switchboardv2 program on-chain.       |
| V1      | <a className="nowrap" href="https://switchboard-xyz.github.io/switchboard-api/"><SiJavascript className="devicons" /> switchboard-api <GoLinkExternal className="devicons" /></a> | A javascript library of utility functions to interact with the Switchboardv1 program.          |
| V1      | <a className="nowrap" href="https://docs.rs/switchboard-program/"><SiRust className="devicons" /> switchboard-program <GoLinkExternal className="devicons" /></a>                 | A rust library of utility functions to interact with the Switchboardv1 program.                |
| V1      | <a className="nowrap" href="https://docs.rs/switchboard-protos/"><SiRust className="devicons" /> switchboard-protos <GoLinkExternal className="devicons" /></a>                   | A rust library of utility functions to interact with the Switchboardv1 program.                |
| V1      | <a className="nowrap" href="https://docs.rs/switchboard-utils/"><SiRust className="devicons" /> switchboard-utils <GoLinkExternal className="devicons" /></a>                     | A rust library of utility functions to interact with the Switchboardv1 program.                |
